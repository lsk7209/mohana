# 히어로 동영상

이 디렉토리에 히어로 섹션에 사용할 동영상 파일을 배치하세요.

## 파일명
- `hero.mp4` - 메인 히어로 섹션 배경 동영상

## 권장 사양
- **형식**: MP4 (H.264 코덱)
- **해상도**: 최소 1920x1080 (Full HD)
- **비율**: 16:9
- **길이**: 10-30초 (루프 재생)
- **파일 크기**: 5MB 이하 권장 (최적화 필요)

## 최적화 팁
1. 동영상 압축 도구 사용 (예: HandBrake, FFmpeg)
2. 해상도는 1920x1080으로 충분 (4K는 불필요)
3. 비트레이트는 2-5 Mbps 권장
4. 프레임레이트는 30fps 권장

## 환경 변수 설정 (선택사항)
동영상 URL을 환경 변수로 설정하려면 `.env.local` 파일에 추가하세요:

```
NEXT_PUBLIC_HERO_VIDEO_URL=/videos/hero.mp4
```

또는 외부 URL을 사용할 수도 있습니다:

```
NEXT_PUBLIC_HERO_VIDEO_URL=https://example.com/videos/hero.mp4
```

## Fallback
동영상 로딩 실패 시 자동으로 이미지로 대체됩니다.
