# 변경 이력

## [0.3.0] - 2025-01-XX

### 추가됨
- ✅ 메시지 상세 보기 완성
  - 메시지 상세 다이얼로그
  - 이벤트 타임라인 표시
  - 오픈/클릭 통계
- ✅ A/B 테스트 기능
  - A/B 테스트 통계 API
  - A/B 테스트 관리 UI
  - 오픈율/클릭율 비교
- ✅ 바운스 처리 및 웹훅
  - 바운스 웹훅 엔드포인트
  - 전달 확인 웹훅
  - 자동 구독 해지 (hard bounce)
- ✅ 시퀀스 성과 분석
  - 시퀀스별 성과 통계
  - 전환율 계산
  - 완료율 추적

### 개선됨
- Admin 네비게이션 헤더 추가
- 리드 상세 페이지에서 메시지 클릭 시 상세 보기
- 페이징 및 CSV 내보내기
- 에러 핸들링 및 입력 검증 강화
- 템플릿 미리보기 기능

## [0.2.0] - 2025-01-XX

### 추가됨
- ✅ 시퀀스 편집 UI 완성
  - 시퀀스 생성/수정 다이얼로그
  - 스텝 추가/삭제 기능
  - 조건부 발송 설정 (미오픈/미클릭)
  - 템플릿 선택 기능
- ✅ 통계 데이터 실제 API 연동
  - 일별 통계 API (`/api/admin/stats/daily`)
  - 개요 통계 API (`/api/admin/stats/overview`)
  - 실시간 데이터 표시
- ✅ 시퀀스 관리 페이지 (`/admin/sequences`)
- ✅ 템플릿 관리 시스템
  - 템플릿 CRUD API
  - 템플릿 목록/편집 페이지
  - 이메일/SMS 템플릿 분리 관리

### 개선됨
- 리드 필터링 및 검색 기능 강화
- 통계 차트 실제 데이터 연동
- 이메일/SMS 발송 다이얼로그 개선
